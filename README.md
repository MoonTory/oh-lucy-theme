# Oh Lucy for Zed

A faithful port of the [Oh Lucy VSCode theme](https://github.com/hermitter/oh-lucy-vscode-theme) by [hermitter](https://github.com/hermitter) to the [Zed](https://zed.dev) editor.

Includes both variants:

- **Oh Lucy**
- **Oh Lucy Evening**

## Development

The original theme is cloned into `reference/oh-lucy-vscode-theme` (git-ignored) and used as the source of truth for all colors.
